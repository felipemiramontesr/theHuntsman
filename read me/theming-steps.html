Drupal 8

Theming steps

  • 1.- Install Drupal 8 (Opcional Enviroment).

  • 2.- Theme setup.

    • 2.1- "mytheme".info.yml

    • 2.2- "mytheme".libraries.yml

  • 3.- Disable Cache and enable Twig Debug.

    • 3.1- In your drupal site go to: your_drupal_site/admin/config/development/performance, and unmark the boxes (Aggregate CSS files and Aggregate JavaScript files) and save the configuration.

    • 3.2- In the same page Clear all caches.

    • 3.3- Disable the cache; copy the files given in the folder "files to copy" into the route: your_drupal_installation/sites/default. Say yes to everything.

    • 3.4- Enable Twig debug; overwriting the file development.services.yml located into the route: your_drupal_installation/sites, with the file given in the folder "files to overwrite". Say yes to everything.

    • 3.5- Clear all caches two or three times just to be secure.

    • 3.6- Refresh your drupal site and open a developer web tools console in your browser. Now you must see the twig debuging working in green font given you diferent tips and suggerations.

    • 3.7- Change the name of your drupal installation in settings.php file at line 799 located at the route: your_drupal_installation\sites\default:

      if (isset($_SERVER['DEVDESKTOP_DRUPAL_SETTINGS_DIR']) && file_exists($_SERVER['DEVDESKTOP_DRUPAL_SETTINGS_DIR'] . '/loc_NAME_OF_YOUR_INSTALLATION_dd.inc')) { require $_SERVER['DEVDESKTOP_DRUPAL_SETTINGS_DIR'] . '/loc_NAME_OF_YOUR_INSTALLATION_dd.inc'; }


    • 3.6- Clear all drupal caches two or three times just to be secure and erase cache browser.
